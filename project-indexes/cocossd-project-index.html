This project aims to combine my interest in machine learning with my current progress in CS 1810.

It uses TensorFlow.js to run the COCO-SSD object detection model.

  • COCO-SSD stands for "Common Ojects in Context - Single Shot Multibox Detection"

The page automatically loads Tensorflow.js and the COCO-SSD model, but waits to let the user use it until it has finished loading. Then, the user can use their webcam to show how the model can detect everyday objects.

Since the class is loaded in the browser, all the data is stored on the local machine. Since there doesn't need to be a server to connect to, the code can run quicker, more securely, and offline.

The code is not originally mine! It is an example from a course by Google that was tweaked afterwards.
