Working with a Weather API

  • This is a project that taught me how to pull data from APIs and format it in order to display on a page.

  • The project uses the VUE framework, combined with bootstrap, to loop through and incrementally add the data to the page.

This project asks the user for permission to use their geolocation and provides a 12-hour forecast for that location. It requests data from the USA National Weather Service in real time, waits for a response, and returns the visual result to the user in a neat little package.
