Working with the Mars Rover API

  • This project was created to help me understand how to gather, sort, and implement data requested from APIs.

  • This project also helped me understand the basics of how how to generically add and format data to a page, without knowing the data given beforehand.

  • I also learned how to gather input from the user with form switches, and how to concatonate the different inputs together to gain a desired output.

The Gist of the Project

  • The program asks the user to choose from a discrete number of options, sends a request to NASA's API for each of the selected options, and concatinates the data together.

  • The data is then filtered and added to the webpage in a uniform way, which allows for any amount of photos to be added to the page.
